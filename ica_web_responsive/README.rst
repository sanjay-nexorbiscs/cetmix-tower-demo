ICA Responsive
==============

ICA Responsive improves the Odoo backend web client with responsive layout,
navbar, home menu, share URL, and dark mode styling assets.

Features
========

* Responsive backend web client customizations.
* Updated navbar and burger menu behavior.
* Home menu layout improvements.
* Share URL tools in the web client.
* Light and dark mode SCSS assets.
* List, kanban, pivot, search, and core UI style improvements.

Configuration
=============

Install the module. The backend assets are applied automatically.

Usage
=====

Use Odoo normally from desktop or smaller screens. The module updates backend
layout and styles through web assets.
